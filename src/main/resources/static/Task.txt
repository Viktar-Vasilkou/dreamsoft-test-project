Разработать REST-сервис, умеющий работать с текстовыми файлами
API сервиса - путь до файла и слово для поиска.
В ответе сервис должен выдать строчки, где найдено слово.
Также, сервис должен запоминать историю поиска в реляционной базе (слово и путь до файла).
Реализовать API отдающее историю (с сортировкой по дате поиска)
 
Стек технологий:
Java 14
Spring boot
Webflux
NIO при работе с файлами
liquibase
